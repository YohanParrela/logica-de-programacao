# 🚀 Lógica de Programação & Algoritmos

Repositório dedicado ao armazenamento de exercícios, desafios de código e estudos de algoritmos desenvolvidos tanto nas aulas da ETEC quanto de forma autônoma (cursos externos, plataformas de desafios e projetos pessoais). 

O objetivo principal é praticar a resolução de problemas, melhorar a eficiência do código e dominar a sintaxe de diferentes linguagens de programação.
